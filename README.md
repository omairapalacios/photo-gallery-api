# Photo Gallery - API con Typescript y Mongo DB

API, desarrollada con Typescript usando Nodejs y Mongodb con el objetivo de guardar archivos de imagen.

### 1.1 API

La API expone los siguientes endpoints:

#### 1.1.1 `/photos`

* `GET /photos`
* `GET /photos/:id`
* `POST /photos`
* `PUT /photos/id`
* `DELETE /photos/:id`
